# Shopcito
Made by:
Yana & Franq

## Descripción

Esta aplicación es una plataforma de e-commerce pensada para pequeños vendedores y negocios independientes que no cuentan con una tienda grande o una infraestructura compleja. Su objetivo es ofrecer un espacio accesible donde cualquier persona pueda crear su propia tienda, publicar productos y promocionarlos de manera sencilla dentro de la plataforma.

Está orientada a la venta de artículos como comida, botanas, productos caseros, accesorios, ropa de segunda mano y otros productos de pequeños emprendimientos o ventas locales.

A diferencia de un e-commerce tradicional, esta plataforma busca adaptarse a las necesidades de cada vendedor, ofreciendo una experiencia más flexible, cercana y personalizable. Cada usuario puede administrar su tienda y contar con un dashboard personalizado según lo que necesite para organizar y mostrar sus productos.
